最近開始自學 Laravel，看過許多資料，覺得以下的內容對我來說相當容易上手，在此記錄下來，也分享給各位。 

## 開始學習 Laravel
我找過許多學習的資料，後來透過IT幫幫忙找到高見龍的影音教學，是我覺得最淺顯易懂的。重點是還不收費！
[為你自己學 Laravel](https://www.youtube.com/playlist?list=PLBd8JGCAcUAFtnWuuqd0tzMwYsVAN4es_)

## 安裝 LNMP 環境
首先我選擇採用自行手動安裝 Linux, Nginx, MySQL, PHP 的方式，而非 Laravel Homestead。  
Install LNMP environment  
[How To Install Linux, Nginx, MySQL, PHP (LEMP stack) on Ubuntu 20.04](https://www.digitalocean.com/community/tutorials/how-to-install-linux-nginx-mysql-php-lemp-stack-on-ubuntu-20-04)

## 安裝 Laravel
環境安裝完成後，就可以直接安裝 Laravel
Install Laravel  
[Installation - Laravel - The PHP Framework For Web Artisans](https://laravel.com/docs/8.x/installation#the-laravel-installer)

## 安裝 Laravel Jetstream - 使用內建的會員系統
想要使用內建的會員系統，必須安裝 Laravel Jetstream 套件
Install member system with Laravel Jetstream
[Installation | Laravel Jetstream](https://jetstream.laravel.com/2.x/installation.html)

**安裝 Node.js 15 可以解決跑 npm 所遇到的錯誤**  
我在安裝完 Laravel Jetstream 後，要跑 npm install & run 時遇到了很多問題。後來我升級 Node.js 到最新版之後就全部解決了！太神奇了！傑克！
Install Node.js 15 will be fixed the npm install error
[How To Install Node.js on Ubuntu 20.04 | DigitalOcean](https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-20-04)

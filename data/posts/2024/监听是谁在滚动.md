### 1. 监听是谁在滚动

​	页面未知原因出现滚动条，揪出来是谁在瞎滚。

```js
function findscroller(element){
  element.onscroll=function () {
    console.log(element)
  }
  Array.from(element.children).forEach(findscroller)
}
findscroller(document.body)
// 把代码片段沾到控制台执行，然后尝试滚动页面也知道谁在动了。
```
